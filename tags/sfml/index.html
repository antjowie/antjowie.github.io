Axios Framework

The framework that powers all of my 2D games

 Posted on June 16, 2018  |  3 minutes (470 words)

The Axios Framework makes use of SFML. The Axios Framework is a 2D game framework that runs on files made by Tiled. The programmer only writes the objects. The objects are exported to a file that can be loaded in Tiled, a 2D tilemap editor. At first, I wanted to make a game engine, but I have no experience making any kind of GUI yet. I believe that you should not do two major new subjects at the same time, so a framework it is. Hopefully, this project can one day evolve into a game engine.
